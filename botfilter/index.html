🧠 Cómo funciona

BotFilter usa un sistema de puntuación heurístico. Cada comentario acumula puntos según distintas señales. Si supera el umbral, es marcado como bot.

Señal Puntos
Frase spam conocida+2
Comentario duplicado+3
URL externa / acortador+1.5
Muy similar a otro comentario+1.5
Comentario genérico muy corto+1.5
Emojis de dinero 💰+1.5
Todo en mayúsculas+1
Exceso de ! o emojis+0.5

🔒 Privacidad

Todo el análisis ocurre directamente en tu dispositivo. No enviamos ningún texto a servidores externos. Sin tracking, sin anuncios, sin cuentas.

🌐 Compatibilidad

Esta web app funciona en cualquier dispositivo sin instalar nada. En computadora también podés usar la extensión de Chrome para filtrado automático.
